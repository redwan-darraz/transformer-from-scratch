# transformer-from-scratch

Rebuilding the Transformer architecture from scratch in Python/PyTorch, following Karpathy's *"Let's build GPT"* lecture and the *"Attention is All You Need"* paper.

## Labs

- **LAB 1.1** — BPE Tokenizer from scratch (encode/decode, comparison with tiktoken)
- **LAB 1.2** — Scaled dot-product attention in NumPy → Multi-Head Attention in PyTorch

## Stack

Python · NumPy · PyTorch · matplotlib
