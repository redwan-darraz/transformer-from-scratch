# LAB 1.2 — Attention Mechanism from Scratch

Implement scaled dot-product attention in NumPy, then port it to Multi-Head Attention in PyTorch.

## Goals

- Implement scaled dot-product attention in NumPy and visualize the heatmap
- Build Multi-Head Attention in PyTorch
- Verify the output matches `nn.MultiheadAttention` (diff < 1e-4)
